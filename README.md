# 法考备考倒计时工具

一款专为法考考生设计的备考倒计时和学习打卡工具，帮助考生清晰把握考试时间，记录每日学习行为，增强备考动力。

## ✨ 特性

- ⏰ **倒计时显示**：实时显示距离2026年法考客观题的剩余天数和小时数
- 💪 **学习打卡**：简单的任务清单，记录每天的学习计划
- 💬 **励志语录**：每次刷新页面显示不同的鼓励语录
- 🎉 **完成祝贺**：完成所有任务时显示鼓励提示
- 🎨 **温暖设计**：学习氛围、鼓励激励、温暖陪伴
- 💾 **无需保存**：纯内存存储，刷新页面清空（专注当下）
- 🚀 **即开即用**：无需安装，打开浏览器即可使用

## 📖 使用说明

### 查看倒计时

打开页面即可看到距离2026年法考客观题的剩余时间：
- 显示格式："还有 XXX 天 XX 小时"
- 每小时自动更新
- 大字号橙色数字，清晰醒目

### 学习打卡

1. **添加任务**：
   - 在输入框中输入学习任务
   - 点击"添加"按钮或按 Enter 键
   - 任务立即显示在列表中

2. **完成任务**：
   - 点击任务文字或复选框
   - 任务显示为已完成状态（绿色背景 + 删除线）
   - 全部完成时显示祝贺提示

3. **删除任务**：
   - 点击任务右侧的删除图标（🗑️）
   - 任务立即删除

### 励志语录

- 每次刷新页面随机显示一条励志语录
- 10条精心挑选的备考语录
- 温柔鼓励，减轻备考压力

## 🎯 设计理念

### 核心价值主张
**"看见时间，记录坚持"** - 用简洁的倒计时唤醒时间意识，用每日打卡强化学习习惯。

### 产品目标
- **时间提醒**：一目了然地看到剩余天数和小时数
- **打卡激励**：通过勾选完成任务获得成就感
- **学习氛围**：营造积极向上的备考氛围
- **简单纯粹**：无数据保存压力，专注当下学习

## 📝 使用建议

### 每日使用流程
1. **早上**：打开页面，查看倒计时，规划今日学习
2. **白天**：完成学习后打卡，获得成就感

- 完成任务后及时打卡，强化成就感
- 刷新页面开始新的一天（清空旧任务）
