# 法考倒计时 2026

极简的 Progressive Web App，为2026年法考考生提供每日倒计时和古典诗词情感支持。

## 功能特点

- 📅 **倒计时**：距离2026年法考还有多少天
- 📜 **每日诗词**：精选古典诗词，每日一首
- 📱 **PWA**：可安装到桌面，支持离线使用
- ⚡ **极速**：3秒内加载，无干扰体验

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview

# 运行测试
npm test
```

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- Vite 5.x（构建工具）
- @vitejs/plugin-pwa（PWA支持）
- Vitest（测试框架）
